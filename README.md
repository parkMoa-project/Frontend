# parkmoa Frontend

## setting
```
Pick a Framework : Vue.js
Package Managers : npm (npm install -g @vue/cli)
CSS Frameworks : Vuetify
```
```
Routing : npm (install vue-router --save)
+ : Axios (npm install axios)
+ : vuex (npm install vuex --save)
```

npm install
